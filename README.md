# 📊 Financial Product Sales Performance Dashboard

## 📌 Project Overview
This project analyzes financial product sales data using Power BI and provides insights into:

- Total Sales
- Total Profit
- Total Units Sold
- Profit Margin
- Sales by Country
- Profit by Segment
- Top Selling Products
- Monthly Sales Trends

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX

## ✨ Dashboard Features
- Interactive slicers
- KPI cards
- Sales trend analysis
- Country-wise and segment-wise insights
